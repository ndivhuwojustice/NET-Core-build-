# NET-Core-build-
A single-page dashboard that visualizes.NET Core Desktop CI builds, test results, artifacts and logs so team can triage failures and ship faster 

Dashboard overview — Build matrix and status pills for quick triage.
Apply filters — Branch/config/job filters to narrow results.
Build details — Test result chart and artifact and log links.
Log preview — Expand logs and copy snippets for debugging.
Upgrade to Pro — Trial offer and team plans (mock UI).
Installable PWA — Works offline and installs as an app.
